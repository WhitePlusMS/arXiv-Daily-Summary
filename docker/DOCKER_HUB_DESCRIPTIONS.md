# Docker Hub 仓库描述

## 仓库 1: arxiv_daily_summary-backend

### Short Description（简短描述）:
```
FastAPI backend for ArXiv daily paper recommendation system with AI-powered summarization and intelligent matching
```

### 中文版本（如果需要）:
```
ArXiv 每日论文推荐系统后端 - 基于 FastAPI 的智能论文推荐与 AI 摘要生成服务
```

### Full Description（完整描述，可选）:
```
ArXiv Daily Paper Recommendation System - Backend API

A FastAPI-based backend service that automatically fetches, analyzes, and summarizes ArXiv papers based on user research interests. Features include:

- Intelligent paper matching using AI models
- Automated daily paper fetching from ArXiv
- AI-powered paper summarization
- RESTful API for frontend integration
- User preference management
- Category-based filtering

Built with Python 3.12, FastAPI, and integrates with DashScope API for AI capabilities.
```

## 仓库 2: arxiv_daily_summary-frontend

### Short Description（简短描述）:
```
Vue3 frontend for ArXiv daily paper recommendation system - modern web interface for paper discovery and management
```

### 中文版本（如果需要）:
```
ArXiv 每日论文推荐系统前端 - 基于 Vue3 的现代化论文发现与管理界面
```

### Full Description（完整描述，可选）:
```
ArXiv Daily Paper Recommendation System - Frontend UI

A modern Vue 3 + TypeScript web application providing an intuitive interface for:

- Daily paper recommendations dashboard
- Research interest configuration
- Paper browsing and filtering
- Environment and API key management
- Real-time progress tracking
- Responsive design for all devices

Built with Vue 3, TypeScript, Vite, and served via Nginx in production.
```
